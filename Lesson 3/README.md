# Day 03 - Floor Plans, Openings, and Layers

**Tutorial:** [Getting Started with Rhino - Part 3](https://www.youtube.com/watch?v=y3BtlybI5cM)

## Topic
Floor Plan Modelling and Layers

## What I Learned
- Used WireCut to cut objects.
- Scaled objects accurately.
- Referenced and traced a floor plan.
- Created doors and windows.
- Created window and door frames.
- Used copy and paste to duplicate objects.
- Measured distances.
- Created and toggled layers.
- Assigned objects to different layers.

## Notes
This lesson focused on using a floor plan as a reference and introduced layers for organising objects within a Rhino model.
