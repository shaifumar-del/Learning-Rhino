# Day 02 - Lines, Faces, and 3D Shapes

**Tutorial:** [Getting Started with Rhino Part 2](https://www.youtube.com/watch?v=MNbkNmNV9J4)

## What I Learned
- Used Ortho for precise movement and drawing.
- Used Auto CPlane and Grid Snap.
- Learned different display and object viewing controls.
- Learned how to cut and modify objects.
- Practised using the Gumball to move and edit objects.

## Notes
This lesson introduced more modelling controls and tools for creating and modifying 3D objects in Rhino.
